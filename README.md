# Wilderness

* A Wilderness pocketmine plugin

## How to Install?
* Put the plugin to you're /plugins folder.
* Restart you're server.
* and done Enjoy.

# Command:
* /wild 
* aliases: /wilderness
* Teleport you to a random location.
* Can be configuate at config.yml
